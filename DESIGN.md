# DESIGN.md — 아그네스 디자인 시스템

> 클린 미니멀 라이트. 밝은 배경, 카드형 UI, 딥그린 포인트.
> 컨셉: *"차고에 있는 나의 애마를 위한, 조용하고 정돈된 정비 노트."*
> 이 문서가 디자인의 Single Source of Truth. 모든 값은 `css/styles.css`의 CSS 변수로 존재한다.

## 1. 디자인 원칙
1. **차분함(Calm).** 화면을 색으로 채우지 않는다. 여백과 위계로 정보를 정리한다.
2. **한눈에 상태 파악.** 대시보드에서 "지금 뭘 교체해야 하나"를 3초 안에 안다. 상태색(정상/주의/교체)이 핵심 언어.
3. **손끝 친화(Touch-first).** 최소 탭 타깃 44px. 하단 탭바로 한 손 조작.
4. **정직한 숫자.** 주행거리·비용·연비는 크고 또렷하게, 단위와 함께.
5. **장식 최소화.** 그림자는 은은하게, 테두리는 얇게, 애니메이션은 짧게(150–220ms).

## 4. 간격 · 반경 · 그림자
- **간격 스케일(px)**: 4, 8, 12, 16, 20, 24, 32 → `--sp-1`..`--sp-7`
- **컨테이너**: 최대폭 `--maxw: 480px` (모바일 앱 폭), 중앙 정렬. 데스크톱에서도 폭 유지 + 넓은 화면은 여백.
- **반경**: `--r-sm 10px`, `--r-md 14px`, `--r-lg 20px`, `--r-pill 999px`
- **그림자**: `--shadow-1: 0 1px 2px rgba(16,24,40,.04), 0 1px 3px rgba(16,24,40,.06)` / `--shadow-2: 0 4px 16px rgba(16,24,40,.08)` (모달/부동버튼)

## 5. 핵심 컴포넌트
- **StatusBar(소모품 상태바)**: 트랙(`--surface-2`) 위에 진행률 채움. 색은 잔여 비율에 따라 ok→warn→danger. 우측에 "남은 N,NNNkm / M개월" 텍스트.
- **Card**: `--surface`, 1px `--border`, `--r-md`, `--shadow-1`, 패딩 16px.
- **Chip/Filter**: pill, 비활성 `--surface-2`/`--text-2`, 활성 `--brand-soft`/`--brand`.
- **하단 탭바(BottomNav)**: 4개(홈·등록·이력·설정). 활성 아이콘/라벨 `--brand`. 안전영역(safe-area-inset) 대응.
- **FAB(+ 등록)**: 딥그린 원형, `--shadow-2`, 하단 탭바 위 우측.
- **Timeline item**: 좌측 카테고리 색 점/아이콘, 날짜·주행거리·비용.
- **가변 입력폼**: 카테고리(정비/주유/세차/부품) 선택에 따라 필드가 바뀜.

## 6. 모션
- 전환: `cubic-bezier(.2,.7,.2,1)`, 160–220ms. 상태바 채움은 300ms ease-out.
- 탭 전환: 페이드+살짝 위로(6px). 과한 슬라이드/바운스 금지.

## 7. 접근성
- 텍스트 대비 WCAG AA. 상태를 색만으로 구분하지 않고 아이콘/텍스트 병기(정상 ✓ / 주의 ! / 교체 ⟳).
- 포커스 링 `2px --brand` 표시. 최소 탭 타깃 44×44.

## 8. 다크 모드
v1은 라이트 전용. 단, 모든 색을 CSS 변수로 두어 후속 `@media (prefers-color-scheme: dark)` 확장이 쉽도록 설계한다.
