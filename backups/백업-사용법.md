# 💾 아그네스 백업 사용설명서

이 폴더(`backups/`)는 앱에서 **내보낸 백업 JSON 파일을 보관하는 곳**입니다.
백업 파일을 여기에 모아두면 나중에 찾기 쉽습니다.

> ⚠️ **중요 — 개인정보 보호**
> 이 저장소는 인터넷에 공개(public)되어 있습니다. 그래서 이 폴더에 넣은 `*.json` 백업 파일은
> **일부러 GitHub에 올라가지 않도록 막아두었습니다**(`.gitignore` 처리). 즉, 백업 파일은
> **내 컴퓨터에만** 남고 인터넷에는 공개되지 않으니 안심하고 여기 보관하세요.
> (더 안전하게 하려면 iCloud/구글드라이브에도 같은 파일을 하나 더 복사해두면 좋습니다.)

---

## 핵심 개념 (딱 3줄)
1. 앱에 입력한 기록은 **자동으로 브라우저 안에 저장**됩니다. 백업을 안 해도 평소엔 안 사라져요.
2. **JSON 내보내기 = 기록 전체를 파일 하나로 복사해두기.** 만일에 대비한 "비상용 사본"입니다.
3. 그 사본을 다시 앱에 넣는 건 **JSON 가져오기**입니다.

---

## ① 내보내기 (백업 뽑기)
1. 앱에서 `설정 → 데이터 백업 → JSON으로 내보내기` 클릭
2. `agnes-backup-2026-07-05.json` 같은 파일이 **다운로드 폴더**에 저장됨
3. 그 파일을 **이 `backups/` 폴더로 옮겨** 보관 (드래그해서 넣으면 끝)

파일 이름 안 바꿔도 됩니다. 날짜가 들어있어서 최신 파일을 알아보기 쉽습니다.

---

## ② 가져오기 (백업 되살리기)
아래 상황일 때만 씁니다:
- 실수로 기록이 지워졌을 때
- 새 폰 / 새 컴퓨터 / 다른 브라우저에서 기록을 이어보고 싶을 때
- 브라우저 캐시를 청소해서 데이터가 날아갔을 때

방법:
1. 앱에서 `설정 → 데이터 백업 → JSON 가져오기` 클릭
2. 파일 선택 창이 뜨면 **이 폴더에 둔 `agnes-backup-....json`** 을 선택
3. "복원할까요?" → 확인 → 끝. 기록이 그대로 되살아납니다.

---

## 언제 백업하나요?
- 정비/주유 기록을 새로 몇 건 넣은 날, 또는 **한 달에 한 번** 정도면 충분합니다.
- 오래된 백업 파일은 지워도 됩니다. **가장 최신 파일 하나**만 있으면 돼요.

---

## 자주 하는 질문
**Q. 이 폴더에 파일을 넣으면 앱에 자동으로 반영되나요?**
A. 아니요. 폴더는 그냥 "보관함"입니다. 반영하려면 반드시 앱에서 `설정 → 가져오기`로 파일을 골라줘야 합니다.

**Q. 백업 파일을 여기 말고 다른 곳에 둬도 되나요?**
A. 네. 다운로드 폴더, iCloud, 구글드라이브 어디든 됩니다. 이 폴더는 "찾기 쉬우라고" 만든 곳일 뿐입니다.

**Q. 이 폴더에 넣은 백업이 인터넷에 공개되나요?**
A. 아니요. `*.json` 파일은 GitHub에 올라가지 않도록 막아두었습니다. 내 컴퓨터에만 남습니다.
