# CLAUDE.md — 아그네스 차량 관리 웹서비스

## 프로젝트 개요
2018 기아 스포티지 2.0 디젤(애칭 "아그네스") **단 한 대**를 위한 개인 차량 관리 웹서비스.
정비 이력 기록, 소모품 교체 시기 예측 알림, 주유/세차/부품 기록, 지출·연비 통계를 제공한다.

- **사용자**: 1인 (차주 본인) 전용
- **아키텍처**: 프론트엔드 전용 (백엔드 없음)
- **배포**: GitHub Pages (`username.github.io/<repo>`)
- **데이터**: 브라우저 `localStorage` + JSON 백업(내보내기/가져오기)

## 기술 스택 (의도적으로 최소화)
- **순수 HTML/CSS/JavaScript (Vanilla, ES Modules)** — 빌드 스텝 없음. GitHub Pages에 파일 그대로 배포.
- **차트**: 외부 라이브러리 없이 자체 SVG 컴포넌트로 구현 (의존성 0, 디자인 완전 통제)
- **폰트**: 시스템 폰트 스택 우선 (Pretendard 계열 → system-ui fallback)
- 프레임워크/번들러/npm 없음. `index.html`을 열면 바로 동작.

## 작업 규칙 (Work Rules)
1. **디자인은 DESIGN.md를 따른다.** 색상·간격·타이포는 CSS 변수로만 사용하고 하드코딩하지 않는다.
2. **모바일 우선(Mobile-first).** 기본 스타일은 모바일 기준, `min-width` 미디어쿼리로 데스크톱을 확장한다. 하단 탭바 = 모바일, 최대폭 컨테이너 중앙 정렬.
3. **데이터 스키마 변경 시 `store.js`에 `SCHEMA_VERSION`을 올리고 마이그레이션을 남긴다.** 사용자 데이터를 절대 조용히 파괴하지 않는다.
4. **주행거리 역전 방지 등 PRD의 예외 처리를 반드시 구현한다.**
5. **모든 텍스트는 한국어.** 통화는 원(₩), 거리 km, 부피 ℓ.
6. **의존성 추가 금지.** 새 라이브러리가 필요하면 먼저 자체 구현을 검토한다.
7. **비밀정보 없음.** 개인정보는 사용자 브라우저에만 저장되며 외부 전송이 없다.
8. **커밋/배포는 사용자 요청 시에만.** 함부로 push 하지 않는다.

## 데이터 저장 원칙
- 단일 키 `agnes_car_data_v1` 하위에 JSON으로 전체 상태 저장.
- 모든 쓰기는 `store.save()`를 경유 (단일 진입점).
- 설정 화면에서 **JSON 내보내기/가져오기**로 기기 이전·백업 가능.
